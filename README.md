# malaria_prevalence_kenya

This repository contains an R script for downloading and visualizing malaria data for Kenya using the Malaria Atlas Project (MAP).

The script automatically downloads malaria prevalence data, processes it for Kenya, and generates visual outputs  showing the spatial distribution and intensity of malaria across the country.

The output data included here are the results generated from running the script.

Files

malaria_kenya.R – R script for downloading, cleaning, and visualizing malaria data.

outputs.csv / outputs.png – Example outputs produced by the script.
